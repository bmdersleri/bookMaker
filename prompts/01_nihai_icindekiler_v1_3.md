## Kitabın genel amacı

Bu kitap, Java ile programlamaya başlayan bilgisayar mühendisliği öğrencilerinin temel sözdizimi, algoritmik düşünme, konsol uygulamaları, koleksiyonlar, hata yönetimi, dosya işlemleri, basit nesne kullanımı, GUI ve başlangıç düzeyi veritabanı uygulamaları geliştirebilecek düzeye gelmesini hedefler.

Kitabın amacı, öğrenciyi yalnızca Java komutlarını ezberleyen biri hâline getirmek değildir. Öğrenci, kitabın sonunda küçük ama tamamlanmış bir masaüstü uygulamasını planlayabilmeli, kodlayabilmeli, test edebilmeli ve raporlayabilmelidir.
