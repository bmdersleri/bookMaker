**JDK (Java Development Kit)** — Java uygulamaları geliştirmek için gerekli derleyici, kütüphane ve araçları içeren yazılım paketi.

**JRE (Java Runtime Environment)** — Java programlarını çalıştırmak için gerekli sanal makine ve temel kütüphaneleri barındıran ortam.

**JVM (Java Virtual Machine)** — Java bytecode’unu işletim sisteminden bağımsız olarak çalıştıran soyut bilgisayar.

**Bytecode** — Java kaynak kodunun derlenmesiyle oluşan, JVM tarafından yorumlanan ara seviyedeki ikili kod.

**Derleyici (Compiler)** — Kaynak kodu makine diline veya bytecode’a çeviren yazılım aracı.

**Sözdizimi (Syntax)** — Bir programlama dilinde geçerli ifadelerin yazım kuralları bütünü.

**Yorum Satırı** — Derleyici tarafından yok sayılan, kodun anlaşılmasını kolaylaştırmak için eklenen açıklama metni.

**REPL (Read-Eval-Print Loop)** — Kodu satır satır yazıp hemen çalıştırmaya olanak tanıyan etkileşimli çalışma ortamı.

**print() Fonksiyonu** — Java’da konsola çıktı yazdırmak için kullanılan temel metot.

**Main Metot** — Java uygulamasının çalıştırılmaya başlandığı, `public static void main(String[] args)` imzasına sahip giriş noktası.

**Kaynak Kod** — Programcının yazdığı, insan tarafından okunabilen Java dosyası (.java uzantılı).

**Derleme** — Kaynak kodun bytecode’a (.class dosyasına) dönüştürülme süreci.

**Platform Bağımsızlık** — Java kodunun bir kez derlenip herhangi bir işletim sisteminde JVM sayesinde çalışabilme özelliği.

**Çalışma Zamanı (Runtime)** — Programın bellekte çalıştığı ve JVM tarafından yönetildiği an.

**Konsol Çıktısı** — Programın çalışması sonucu terminal veya komut satırında görüntülenen metin.
