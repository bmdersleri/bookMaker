# Bölüm Özel Üretim Promptu — Performans, Yayınlama ve Dağıtım

Bu dosya, `performans-yayinlama` bölümü için `prompts/default_chapter.md` üzerine ek bağlam sağlar.

## Bölüm amacı

Okuyucu debug/profile/release build farkını açıklayabilmeli.

## Bu bölümde özellikle korunacak kapsam

- Debug build
- Profile mode
- Release build
- APK
- AAB
- App signing
- Versioning
- Performance profiling

## Mini uygulama

**Yayınlamaya hazır örnek uygulama kontrolü**

## Bölüm dışına taşmaması gereken konular

- App Store Connect ileri ayrıntılar
- Play Store optimizasyonu
- Kurumsal imzalama senaryoları

## Üretim notu

- `chapter_manifest.yaml` içindeki `structure.sections` sırasını takip et.
- En az 1 ekran çıktısı placeholderı kullan.
- Kodlar çalıştırılabilir değilse `validation_mode: review_only` olarak işaretlenmelidir.
- Bölüm sonunda laboratuvar görevi ve değerlendirme rubriği mutlaka bulunmalıdır.
