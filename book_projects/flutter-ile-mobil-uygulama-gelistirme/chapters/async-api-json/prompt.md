# Bölüm Özel Üretim Promptu — Async Programlama, REST API ve JSON

Bu dosya, `async-api-json` bölümü için `prompts/default_chapter.md` üzerine ek bağlam sağlar.

## Bölüm amacı

Okuyucu REST API’den veri çekebilmeli.

## Bu bölümde özellikle korunacak kapsam

- Future
- async/await
- HTTP
- REST
- JSON
- Model mapping
- FutureBuilder
- Error handling

## Mini uygulama

**API destekli katalog uygulaması**

## Bölüm dışına taşmaması gereken konular

- GraphQL
- WebSocket
- OAuth ileri ayrıntılar

## Üretim notu

- `chapter_manifest.yaml` içindeki `structure.sections` sırasını takip et.
- En az 2 ekran çıktısı placeholderı kullan.
- Kodlar çalıştırılabilir değilse `validation_mode: review_only` olarak işaretlenmelidir.
- Bölüm sonunda laboratuvar görevi ve değerlendirme rubriği mutlaka bulunmalıdır.
