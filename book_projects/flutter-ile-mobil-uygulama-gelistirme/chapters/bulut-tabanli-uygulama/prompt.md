# Bölüm Özel Üretim Promptu — Firebase/Supabase ile Bulut Tabanlı Uygulama

Bu dosya, `bulut-tabanli-uygulama` bölümü için `prompts/default_chapter.md` üzerine ek bağlam sağlar.

## Bölüm amacı

Okuyucu bulut servis kullanım senaryolarını açıklayabilmeli.

## Bu bölümde özellikle korunacak kapsam

- Firebase
- Supabase
- Authentication
- Cloud database
- Security rules
- Environment configuration
- Offline/hata durumları

## Mini uygulama

**Bulut destekli kayıt takip uygulaması**

## Bölüm dışına taşmaması gereken konular

- İleri backend güvenliği
- Cloud Functions ileri kullanım
- Ödeme sistemleri

## Üretim notu

- `chapter_manifest.yaml` içindeki `structure.sections` sırasını takip et.
- En az 2 ekran çıktısı placeholderı kullan.
- Kodlar çalıştırılabilir değilse `validation_mode: review_only` olarak işaretlenmelidir.
- Bölüm sonunda laboratuvar görevi ve değerlendirme rubriği mutlaka bulunmalıdır.
