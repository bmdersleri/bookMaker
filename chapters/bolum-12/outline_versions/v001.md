## H1: Java 8+ ile Tarih ve Zaman Yönetimi

### H2: 1. Tarih ve Zaman API'sine Giriş: `java.time` Paketi
- **Kavram:** Eski `Date` ve `Calendar` sınıflarının sorunları (thread güvenliği eksikliği, mutable yapı, zaman dilimi karmaşası)
- **Kavram:** Java 8 ile gelen `java.time` paketinin temel felsefesi (immutable, thread-safe, ISO 8601 uyumlu)
- **Örnek:** Eski ve yeni API arasındaki farkı gösteren basit bir kod karşılaştırması
  ```java
  // Eski yöntem
  Date eskiTarih = new Date();
  System.out.println(eskiTarih);
  
  // Yeni yöntem
  LocalDate yeniTarih = LocalDate.now();
  System.out.println(yeniTarih);
  ```
- **Uygulama:** Konsola şu anki tarihi yeni API ile yazdıran bir program yazın

### H2: 2. Temel Sınıflar: `LocalDate`, `LocalTime`, `LocalDateTime`
- **Kavram:** 
  - `LocalDate`: Sadece tarih (yıl, ay, gün) içerir, zaman bilgisi yok
  - `LocalTime`: Sadece zaman (saat, dakika, saniye, nanosaniye) içerir, tarih bilgisi yok
  - `LocalDateTime`: Hem tarih hem zaman bilgisini bir arada tutar
- **Örnek:** Her üç sınıfın oluşturulması ve temel metotlarının kullanımı
  ```java
  LocalDate bugun = LocalDate.now();
  LocalTime suAn = LocalTime.now();
  LocalDateTime bugunVeSaat = LocalDateTime.now();
  
  LocalDate ozelTarih = LocalDate.of(2023, Month.DECEMBER, 25);
  LocalTime ozelZaman = LocalTime.of(14, 30, 0);
  LocalDateTime ozelZamanliTarih = LocalDateTime.of(ozelTarih, ozelZaman);
  
  System.out.println("Bugün: " + bugun);
  System.out.println("Şu an: " + suAn);
  System.out.println("Özel tarih: " + ozelTarih);
  ```
- **Uygulama:** Kullanıcıdan doğum tarihi (yıl, ay, gün) ve doğum saati (saat, dakika) alarak `LocalDateTime` nesnesi oluşturan bir program yazın
- **Değerlendirme:** Aşağıdaki senaryolarda hangi sınıf kullanılmalıdır?
  - Bir uçuşun kalkış saati
  - Bir kişinin doğum günü
  - Bir etkinliğin başlangıç zamanı

### H2: 3. Tarih/Zaman Biçimlendirme ve Ayrıştırma: `DateTimeFormatter`
- **Kavram:** Tarih/zaman nesnelerini metne çevirme (formatlama) ve metinden tarih/zaman nesnesi oluşturma (parse etme)
- **Kavram:** Önceden tanımlı formatlar (`ISO_LOCAL_DATE`, `ISO_DATE_TIME`) ve özel desen oluşturma (`yyyy/MM/dd`, `dd-MM-yyyy HH:mm`)
- **Örnek:** Farklı formatlarda tarih yazdırma ve string'den tarih oluşturma
  ```java
  LocalDateTime simdi = LocalDateTime.now();
  
  // Önceden tanımlı format
  String isoFormat = simdi.format(DateTimeFormatter.ISO_LOCAL_DATE_TIME);
  System.out.println("ISO format: " + isoFormat);
  
  // Özel desen
  DateTimeFormatter ozelFormat = DateTimeFormatter.ofPattern("dd/MM/yyyy - HH:mm:ss");
  String ozelFormatli = simdi.format(ozelFormat);
  System.out.println("Özel format: " + ozelFormatli);
  
  // String'den tarih oluşturma (parse)
  String tarihMetni = "25-12-2023 14:30";
  DateTimeFormatter parseFormat = DateTimeFormatter.ofPattern("dd-MM-yyyy HH:mm");
  LocalDateTime ayrismaTarihi = LocalDateTime.parse(tarihMetni, parseFormat);
  System.out.println("Ayrıştırılan tarih: " + ayrismaTarihi);
  ```
- **Uygulama:** Kullanıcıdan "gg.AA.yyyy HH:mm" formatında bir tarih metni alın, bunu `LocalDateTime` nesnesine çevirin ve "yyyy/MM/dd - HH:mm" formatında ekrana yazdırın
- **Değerlendirme:** `DateTimeFormatter.ofPattern()` içinde kullanılan harflerin (y, M, d, H, m, s) anlamlarını ve tekrar sayılarının (yy vs yyyy) farkını açıklayın

### H2: 4. Tarih ve Zaman Hesaplamaları: Ekleme, Çıkarma ve Fark Bulma
- **Kavram:** `plus` ve `minus` metotları ile tarih/zaman üzerinde aritmetik işlemler
- **Kavram:** `Period` (tarih farkı) ve `Duration` (zaman farkı) sınıfları
- **Örnek:** Gün, ay, yıl ekleme/çıkarma ve iki tarih arasındaki farkı hesaplama
  ```java
  LocalDate bugun = LocalDate.now();
  LocalDate birHaftaSonra = bugun.plusWeeks(1);
  LocalDate ucAyOnce = bugun.minusMonths(3);
  
  LocalDate dogumGunu = LocalDate.of(1990, Month.MARCH, 15);
  Period yas = Period.between(dogumGunu, bugun);
  System.out.println("Yaş: " + yas.getYears() + " yıl " + 
                     yas.getMonths() + " ay " + yas.getDays() + " gün");
  
  LocalTime baslangic = LocalTime.of(9, 30);
  LocalTime bitis = LocalTime.of(17, 45);
  Duration calismaSuresi = Duration.between(baslangic, bitis);
  System.out.println("Çalışma süresi: " + calismaSuresi.toHours() + " saat " + 
                     calismaSuresi.toMinutesPart() + " dakika");
  ```
- **Uygulama:** Kullanıcıdan iki farklı tarih (gün, ay, yıl) alın, aralarındaki farkı yıl, ay, gün cinsinden hesaplayan ve ekrana yazdıran bir program yazın
- **Değerlendirme:** `Period` ve `Duration` arasındaki temel fark nedir? Hangi durumda hangisini kullanırsınız?

### H2: 5. Zaman Dilimleri ve Gelecek/Zamanlanmış İşlemler
- **Kavram:** `ZonedDateTime` ile zaman dilimi desteği, `ZoneId` kullanımı
- **Kavram:** `ChronoUnit` ile daha esnek zaman hesaplamaları
- **Örnek:** Farklı zaman dilimlerinde anlık zamanı gösterme ve belirli bir tarihe kalan süreyi hesaplama
  ```java
  // Zaman dilimleri
  ZonedDateTime istanbul = ZonedDateTime.now(ZoneId.of("Europe/Istanbul"));
  ZonedDateTime newYork = ZonedDateTime.now(ZoneId.of("America/New_York"));
  System.out.println("İstanbul: " + istanbul);
  System.out.println("New York: " + newYork);
  
  // ChronoUnit ile hesaplama
  LocalDate yilbasi = LocalDate.of(2025, Month.JANUARY, 1);
  LocalDate bugun = LocalDate.now();
  long kalanGun = ChronoUnit.DAYS.between(bugun, yilbasi);
  long kalanAy = ChronoUnit.MONTHS.between(bugun, yilbasi);
  System.out.println("Yılbaşına kalan gün: " + kalanGun);
  System.out.println("Yılbaşına kalan ay: " + kalanAy);
  
  // Tarih kontrolü
  boolean onceMi = bugun.isBefore(yilbasi);
  boolean sonraMi = bugun.isAfter(yilbasi);
  System.out.println("Yılbaşından önce mi? " + onceMi);
  ```
- **Uygulama:** Kullanıcıdan bir hedef tarih (gün, ay, yıl) alın, bugünden itibaren bu tarihe kaç gün, kaç saat kaldığını hesaplayan bir program yazın. Ayrıca İstanbul ve Tokyo saat dilimlerinde şu anki zamanı gösterin

  - `ChronoUnit` ile `Period`/`Duration` arasındaki farkı açıklayın
