# Java’ya Giriş, Çalışma Modeli ve Geliştirme Ortamı

## 1. Java Nedir ve Neden Kullanılır?
### 1.1. Java’nın Tarihçesi ve Gelişimi
- James Gosling ve Sun Microsystems dönemi
- Java’nın evrimi: Java 1.0’dan günümüze

### 1.2. Java’nın Kullanım Alanları
- Web uygulamaları (Spring, Java EE)
- Mobil uygulamalar (Android)
- Kurumsal yazılımlar
- Büyük veri ve yapay zeka (Apache Hadoop, Spark)
- Gömülü sistemler ve IoT

### 1.3. Java’nın Avantajları
- Platform bağımsızlık (Write Once, Run Anywhere)
- Nesne yönelimli programlama
- Güçlü bellek yönetimi (Garbage Collection)
- Geniş kütüphane ve topluluk desteği

## 2. Java’nın Çalışma Modeli: JVM, JRE ve JDK
### 2.1. JDK (Java Development Kit)
- Derleyici (javac), yorumlayıcı, hata ayıklayıcı
- Geliştirme araçları ve kütüphaneler
- **Kod Örneği:** JDK kurulumunu doğrulama komutu (`java -version`, `javac -version`)

### 2.2. JRE (Java Runtime Environment)
- Çalışma zamanı kütüphaneleri
- JVM’i içeren paket
- **Kod Örneği:** Basit bir Java programını JRE ile çalıştırma

### 2.3. JVM (Java Virtual Machine)
- Bytecode’un yorumlanması ve çalıştırılması
- Bellek yönetimi ve Garbage Collection
- Platform bağımsızlığın temel taşı
- **Kod Örneği:** Bytecode üretme süreci (`javac HelloWorld.java` → `java HelloWorld`)

### 2.4. JDK, JRE ve JVM Arasındaki İlişki
- Katmanlı mimari şeması
- Hangi durumda hangi bileşen kullanılır?

## 3. Java Geliştirme Ortamı Kurulumu
### 3.1. İşletim Sistemine Göre Kurulum
- Windows’a JDK kurulumu
- macOS’a JDK kurulumu
- Linux’a JDK kurulumu (APT, YUM)

### 3.2. PATH ve JAVA_HOME Değişkenleri
- PATH değişkenine JDK ekleme
- JAVA_HOME ayarlama
- **Kod Örneği:** Komut satırında kurulum doğrulama

### 3.3. İntegre Geliştirme Ortamları (IDE)
- Eclipse, IntelliJ IDEA, NetBeans
- IDE seçim kriterleri ve kurulum

## 4. İlk Java Programı: “Merhaba Dünya”
### 4.1. Kaynak Kod Yazma
- .java dosyası yapısı
- Class tanımı ve main metodu
- **Kod Örneği:**
```java
public class MerhabaDunya {
    public static void main(String[] args) {
        System.out.println("Merhaba Dünya!");
    }
}
```

### 4.2. Derleme ve Çalıştırma Süreci
- `javac` ile bytecode üretme
- `java` ile çalıştırma
- **Kod Örneği:** Derleme ve çalıştırma komutları

### 4.3. Hata Ayıklama ve Çıktı Analizi
- Derleme hataları ve anlamları
- Çalışma zamanı hataları
- **Uygulama:** Farklı hata türleriyle karşılaşma ve çözüm

## 5. Java Programlama Süreci ve İyi Uygulamalar
### 5.1. Kaynak Kod Düzeni
- Dosya adı ile class adı eşleşmesi
- Paket yapısı ve isimlendirme kuralları

### 5.2. Derleme-Çalıştırma Döngüsü
- Kod yaz → Derle → Çalıştır → Hata ayıkla
- **Uygulama:** Basit bir hesap makinesi programı yazma

### 5.3. Temel Hata Ayıklama Teknikleri
- System.out.println ile izleme
- IDE hata ayıklayıcı kullanımı
- **Değerlendirme:** Verilen hatalı kodu düzeltme alıştırması

### 5.4. Java Dokümantasyonu ve Kaynaklar
- Oracle Java Dokümantasyonu
- Java API dokümantasyonu okuma
- Topluluk kaynakları (Stack Overflow, Baeldung)
