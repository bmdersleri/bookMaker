# Koleksiyonlar ve Dinamik Veri Yönetimi

### 1.2 Koleksiyon Türlerine Genel Bakış
- List, Set, Map arayüzleri ve temel özellikleri

## 2. ArrayList: Dinamik Dizi Yapısı
### 2.1 ArrayList'in Çalışma Prensibi
- İç yapısı ve kapasite yönetimi
- Eleman ekleme/silme işlemlerinin karmaşıklığı
- **Kod Örneği: ArrayList oluşturma ve temel işlemler**
```java
ArrayList<String> ogrenciler = new ArrayList<>();
ogrenciler.add("Ali");
ogrenciler.add("Ayşe");
ogrenciler.add(1, "Mehmet"); // Belirli indexe ekleme
String silinen = ogrenciler.remove(0);
```

### 2.2 ArrayList ile İleri Düzey İşlemler
- Döngülerle eleman işleme (for, for-each, iterator)
- Liste dönüşümleri ve alt liste oluşturma
- **Uygulama: Öğrenci not ortalaması hesaplama**
```java
ArrayList<Integer> notlar = new ArrayList<>(Arrays.asList(85, 92, 78, 95));
double ortalama = notlar.stream()
    .mapToInt(Integer::intValue)
    .average()
    .orElse(0.0);
```

## 3. LinkedList: Bağlı Liste Yapısı
### 3.1 LinkedList'in Yapısı ve Özellikleri
- Node yapısı ve çift yönlü bağlantı
- ArrayList ile performans karşılaştırması
- **Kod Örneği: LinkedList oluşturma ve kuyruk işlemleri**
```java
LinkedList<String> kuyruk = new LinkedList<>();
kuyruk.offer("İlk"); // Kuyruğa ekle
kuyruk.offer("İkinci");
String ilk = kuyruk.poll(); // Kuyruktan çıkar
String bas = kuyruk.peek(); // Sadece görüntüle
```

### 3.2 LinkedList ile Stack ve Queue İşlemleri
- Deque arayüzü ve çift yönlü işlemler
- LIFO (Stack) ve FIFO (Queue) prensipleri
- **Uygulama: İşlem geçmişi yönetimi (Undo/Redo)**
```java
LinkedList<String> islemGecmisi = new LinkedList<>();
islemGecmisi.push("Dosya açıldı"); // Stack'e ekle
islemGecmisi.push("Düzenleme yapıldı");
String geriAl = islemGecmisi.pop(); // Son işlemi al
```

## 4. HashMap: Anahtar-Değer Eşlemesi
### 4.1 HashMap'in Çalışma Prensibi
- Hash fonksiyonu ve bucket yapısı
- Çakışma (collision) çözümleme yöntemleri
- **Kod Örneği: HashMap oluşturma ve temel işlemler**
```java
HashMap<String, Integer> ogrenciNotlari = new HashMap<>();
ogrenciNotlari.put("Ali", 85);
ogrenciNotlari.put("Ayşe", 92);
int aliNotu = ogrenciNotlari.getOrDefault("Ali", 0);
ogrenciNotlari.remove("Mehmet");
```

### 4.2 HashMap ile Veri Yönetimi
- Key ve value setlerini dolaşma
- HashMap performans optimizasyonu (initial capacity, load factor)
- **Uygulama: Kelime sayacı uygulaması**
```java
String metin = "java java programlama dili";
HashMap<String, Integer> kelimeSayaci = new HashMap<>();
for (String kelime : metin.split(" ")) {
    kelimeSayaci.merge(kelime, 1, Integer::sum);
}
```

## 5. HashSet: Benzersiz Eleman Koleksiyonu
### 5.1 HashSet'in Yapısı ve Kullanımı
- Set arayüzü ve matematiksel küme özellikleri
- HashSet'in HashMap ile ilişkisi
- **Kod Örneği: HashSet oluşturma ve küme işlemleri**
```java
HashSet<String> ogrenciSet = new HashSet<>();
ogrenciSet.add("Ali");
ogrenciSet.add("Ayşe");
ogrenciSet.add("Ali"); // Tekrar eklenmez!
boolean varMi = ogrenciSet.contains("Ali");
```

### 5.2 HashSet ile Küme Operasyonları
- Birleşim, kesişim, fark işlemleri
- HashSet performansı ve hashCode/equals metotları
- **Uygulama: Benzersiz IP adreslerini takip etme**
```java
HashSet<String> benzersizIP = new HashSet<>();
String[] ipListesi = {"192.168.1.1", "192.168.1.2", "192.168.1.1"};
Collections.addAll(benzersizIP, ipListesi);
System.out.println("Benzersiz IP sayısı: " + benzersizIP.size());
```

## 6. Koleksiyon Algoritmaları ve İleri Düzey İşlemler
### 6.1 Collections Sınıfı Yardımcı Metotları
- Sıralama (sort), karıştırma (shuffle), ters çevirme (reverse)
- Arama (binarySearch), kopyalama (copy), doldurma (fill)
- **Kod Örneği: Collections sınıfı kullanımı**
```java
ArrayList<Integer> sayilar = new ArrayList<>(Arrays.asList(5, 2, 8, 1, 9));
Collections.sort(sayilar);
Collections.reverse(sayilar);
int index = Collections.binarySearch(sayilar, 5);
Collections.shuffle(sayilar);
```

### 6.2 Koleksiyonların Dönüşümü ve Filtrelenmesi
- Stream API ile koleksiyon işlemleri
- Filtreleme, dönüştürme, gruplama
- **Uygulama: Stream API ile veri analizi**
```java
List<Integer> ciftSayilar = sayilar.stream()
    .filter(s -> s % 2 == 0)
    .map(s -> s * 2)
    .collect(Collectors.toList());
```

### 6.3 Thread Güvenli Koleksiyonlar
- synchronized koleksiyonlar
- ConcurrentHashMap ve CopyOnWriteArrayList
- **Değerlendirme: Performans karşılaştırması ve doğru koleksiyon seçimi**
```java
// Farklı koleksiyon türlerinin performans testi
long baslangic = System.nanoTime();
// ... koleksiyon işlemleri ...
long bitis = System.nanoTime();
System.out.println("Geçen süre: " + (bitis - baslangic) + " ns");
```
