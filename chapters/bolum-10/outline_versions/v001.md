# String İşlemleri ve Metin Problemleri

## Giriş
Bu bölümde, programlamanın temel yapı taşlarından biri olan string (metin) işlemlerini ve metin manipulasyonu tekniklerini öğreneceksiniz. String sınıfı metodları, karakter işlemleri ve metin problemlerinin çözümü için gerekli tüm kavramları adım adım inceleyeceğiz.

## H2: String Sınıfı ve Temel Metodlar

### String Oluşturma ve İlk Değer Atama
- String değişken tanımlama yöntemleri
- Boş string kontrolü
- String literal vs String nesnesi farkı

**Kod Örneği:**
```python
# String oluşturma örnekleri
isim = "Ahmet"
metin = str("Merhaba Dünya")
bos_string = ""

print(f"İsim: {isim}")
print(f"Metin: {metin}")
print(f"Boş mu?: {len(bos_string) == 0}")
```

### String Uzunluğu ve İndeksleme
- len() fonksiyonu kullanımı
- Pozitif ve negatif indeksleme
- Karakter erişimi

**Kod Örneği:**
```python
metin = "Python Programlama"
print(f"Uzunluk: {len(metin)}")
print(f"İlk karakter: {metin[0]}")
print(f"Son karakter: {metin[-1]}")
print(f"3. indeks: {metin[3]}")
```

### String Dilimleme (Slicing)
- [başlangıç:bitiş:adım] syntax'ı
- Ters çevirme işlemleri
- Alt string çıkarma

**Kod Örneği:**
```python
metin = "Merhaba Dünya"
print(f"İlk 7 karakter: {metin[:7]}")
print(f"Son 5 karakter: {metin[-5:]}")
print(f"Ters çevrilmiş: {metin[::-1]}")
print(f"2. indeksten 8. indekse: {metin[2:8]}")
```

## H2: String Metin Manipülasyon Metodları

### Büyük/Küçük Harf Dönüşümleri
- upper(), lower(), capitalize(), title()
- swapcase() kullanımı
- casefold() ile özel karakter dönüşümleri

**Kod Örneği:**
```python
metin = "pYthon prOgramlama DİLİ"
print(f"Büyük harf: {metin.upper()}")
print(f"Küçük harf: {metin.lower()}")
print(f"Başlık formatı: {metin.title()}")
print(f"İlk harf büyük: {metin.capitalize()}")
print(f"Swapcase: {metin.swapcase()}")
```

### Boşluk ve Karakter Temizleme
- strip(), lstrip(), rstrip() metodları
- replace() ile karakter değiştirme
- translate() ile toplu dönüşüm

**Kod Örneği:**
```python
metin = "   ***Merhaba Dünya***   "
print(f"Strip: '{metin.strip()}'")
print(f"Lstrip: '{metin.lstrip()}'")
print(f"Rstrip: '{metin.rstrip()}'")
print(f"Replace: '{metin.replace('*', '')}'")

# Özel karakter temizleme
temizlenecek = "  \t\n  Test Metni  \n\t  "
print(f"Temizlenmiş: '{temizlenecek.strip()}'")
```

### Bölme ve Birleştirme İşlemleri
- split() ile string bölme
- join() ile liste birleştirme
- partition() ve rpartition() kullanımı

**Kod Örneği:**
```python
metin = "elma,armut,muz,çilek"
meyveler = metin.split(",")
print(f"Bölünmüş liste: {meyveler}")

birlesik = " - ".join(meyveler)
print(f"Birleştirilmiş: {birlesik}")

# Partition örneği
telefon = "+90-555-123-4567"
print(f"Partition: {telefon.partition('-')}")
print(f"Rpartition: {telefon.rpartition('-')}")
```

## H2: String Arama ve Kontrol Metodları

### İçerik Kontrol Metodları
- startswith(), endswith() kullanımı
- in operatörü ile arama
- count() ile tekrar sayısı bulma

**Kod Örneği:**
```python
metin = "Python programlama dili Python ile yazılım geliştirme"
print(f"'Python' ile başlıyor mu?: {metin.startswith('Python')}")
print(f"'geliştirme' ile bitiyor mu?: {metin.endswith('geliştirme')}")
print(f"'Java' metinde var mı?: {'Java' in metin}")
print(f"'Python' kaç kez geçiyor?: {metin.count('Python')}")
```

### Bulma ve İndeks Metodları
- find(), index(), rfind(), rindex()
- Hata durumları ve farkları
- in operatörü ile güvenli arama

**Kod Örneği:**
```python
metin = "Merhaba Dünya, Merhaba Evren"
print(f"İlk 'Merhaba' indeksi: {metin.find('Merhaba')}")
print(f"Son 'Merhaba' indeksi: {metin.rfind('Merhaba')}")
print(f"Bulunamayan kelime: {metin.find('Selam')}")  # -1 döner

try:
    print(f"Index ile: {metin.index('Selam')}")
except ValueError as e:
    print(f"Hata: {e}")
```

### Karakter Tipi Kontrolleri
- isalpha(), isdigit(), isalnum()
- isspace(), islower(), isupper()
- isdecimal(), isnumeric() farkları

**Kod Örneği:**
```python
testler = ["Python3", "12345", "   ", "Merhaba", "3.14"]
for test in testler:
    print(f"'{test}' -> alpha: {test.isalpha()}, digit: {test.isdigit()}, "
          f"alnum: {test.isalnum()}, space: {test.isspace()}")

# Özel durumlar
print(f"'Ⅻ' isnumeric: {'Ⅻ'.isnumeric()}")  # Roma rakamı
print(f"'²' isdigit: {'²'.isdigit()}")      # Üst simge
```

## H2: String Biçimlendirme ve Formatlama

### f-string Kullanımı
- Değişken ekleme
- İfade kullanımı
- Hizalama ve dolgu

**Kod Örneği:**
```python
isim = "Ali"
yas = 25
maas = 5250.50

print(f"Ad: {isim}, Yaş: {yas}, Maaş: {maas:.2f} TL")
print(f"Sağa hizalı: {isim:>10}")
print(f"Sola hizalı: {isim:<10}")
print(f"Ortala: {isim:^10}")
print(f"Dolgulu: {isim:*^10}")
```

### format() Metodu
- Konumsal parametreler
- Anahtarlı parametreler
- Sözlük ile kullanım

**Kod Örneği:**
```python
# Konumsal parametreler
print("{} {} {} yılında doğdu.".format("Ahmet", "1980", "İstanbul"))

# Anahtarlı parametreler
print("{ad} {soyad} - {meslek}".format(
    ad="Ayşe", soyad="Yılmaz", meslek="Mühendis"))

# Sözlük ile kullanım
kisi = {"ad": "Mehmet", "yas": 30}
print("Ad: {ad}, Yaş: {yas}".format(**kisi))
```

### % Operatörü ile Eski Tip Formatlama
- %s, %d, %f kullanımı
- Ondalık hassasiyeti
- Çoklu parametreler

**Kod Örneği:**
```python
isim = "Veli"
not_ort = 85.6789

print("Öğrenci: %s, Not ortalaması: %.2f" % (isim, not_ort))
print("Sayı: %+d" % 42)
print("Ondalık: %.3f" % 3.14159)
```

## H2: Gelişmiş String İşlemleri ve Problem Çözme

### Düzenli İfadelerle String İşleme
- re modülü temel fonksiyonları
- Desen eşleştirme
- Metin temizleme ve dönüştürme

**Kod Örneği:**
```python
import re

metin = "Telefon: 555-123-4567, E-posta: test@ornek.com"
telefon_deseni = r'\d{3}-\d{3}-\d{4}'
eposta_deseni = r'\w+@\w+\.\w+'

telefon = re.search(telefon_deseni, metin)
eposta = re.search(eposta_deseni, metin)

print(f"Bulunan telefon: {telefon.group() if telefon else 'Bulunamadı'}")
print(f"Bulunan e-posta: {eposta.group() if eposta else 'Bulunamadı'}")

# Metin temizleme
kirli_metin = "Merhaba!! Bu bir *** test ## metnidir."
temiz = re.sub(r'[^\w\s]', '', kirli_metin)
print(f"Temizlenmiş: {temiz}")
```

### String Kodlama ve Karakter Setleri
- encode() ve decode() metodları
- UTF-8, ASCII, Latin-1 dönüşümleri
- Unicode karakter işlemleri

**Kod Örneği:**
```python
metin = "Merhaba Dünya 🌍"

# Encoding
utf8_bytes = metin.encode('utf-8')
print(f"UTF-8 bytes: {utf8_bytes}")

# Decoding
cozulmus = utf8_bytes.decode('utf-8')
print(f"Çözülmüş: {cozulmus}")

# ASCII kontrol
try:
    metin.encode('ascii')
except UnicodeEncodeError as e:
    print(f"ASCII'ye dönüştürülemez: {e}")
```

### String Performans İpuçları
- Büyük metinlerde join() vs + operatörü
- StringBuilder benzeri yaklaşımlar
- Bellek yönetimi

**Kod Örneği:**
```python
import time

# Kötü performans - + operatörü
baslangic = time.time()
sonuc = ""
for i in range(10000):
    sonuc += str(i)
bitis = time.time()
print(f"+ operatörü: {bitis - baslangic:.4f} saniye")

# İyi performans - join() metodu
baslangic = time.time()
parcalar = [str(i) for i in range(10000)]
sonuc = "".join(parcalar)
bitis = time.time()
print(f"join() metodu: {bitis - baslangic:.4f} saniye")
```

### Pratik Problem Çözümleri
- Palindrom kontrolü
- Anagram tespiti
- Metin sıkıştırma
- Şifreleme işlemleri

**Kod Örneği:**
```python
# Palindrom kontrolü
def palindrom_mu(metin):
    metin = metin.lower().replace(" ", "")
    return metin == metin[::-1]

print(f"'kabak' palindrom mu?: {palindrom_mu('kabak')}")
print(f"'Merhaba' palindrom mu?: {palindrom_mu('Merhaba')}")

# Anagram tespiti
def anagram_mi(kelime1, kelime2):
    return sorted(kelime1.lower()) == sorted(kelime2.lower())

print(f"'elma' ve 'lame' anagram mı?: {anagram_mi('elma', 'lame')}")
print(f"'kitap' ve 'katip' anagram mı?: {anagram_mi('kitap', 'katip')}")

# Basit metin sıkıştırma
def metin_sikistir(metin):
    if not metin:
        return ""
    
    sonuc = []
    sayac = 1
    
    for i in range(1, len(metin)):
        if metin[i] == metin[i-1]:
            sayac += 1
        else:
            sonuc.append(f"{metin[i-1]}{sayac}")
            sayac = 1
    
    sonuc.append(f"{metin[-1]}{sayac}")
    return "".join(sonuc)

print(f"'aaabbbcc' sıkıştırılmış: {metin_sikistir('aaabbbcc')}")
print(f"'abc' sıkıştırılmış: {metin_sikistir('abc')}")
```
