# Java’da Dosya İşlemleri ve Kalıcı Veri Saklama

## 1. Temel Dosya İşlemleri: `File` Sınıfı ve Dosya Sistemi ile Etkileşim
- **Kavram**: `java.io.File` sınıfının amacı, dosya ve dizin yollarını temsil etmek; dosya varlığını kontrol etme, oluşturma, silme ve metaveri erişimi.
- **Örnek**: Bir dosyanın var olup olmadığını kontrol etme, yeni bir dizin oluşturma.
- **Uygulama** *(kod örneği)*:
  ```java
  File file = new File("ornek.txt");
  if (!file.exists()) {
      file.createNewFile();
      System.out.println("Dosya oluşturuldu: " + file.getAbsolutePath());
  }
  System.out.println("Dosya boyutu: " + file.length() + " bayt");
  ```
- **Değerlendirme**: `File` sınıfının sınırlamaları (yalnızca metaveri, içerik okuma/yazma yok) ve neden daha gelişmiş sınıflara ihtiyaç duyulduğu.

## 2. Karakter Tabanlı Okuma/Yazma: `FileReader` ve `FileWriter`
- **Kavram**: Karakter akışları (character streams) ile metin dosyalarına okuma/yazma işlemleri; `FileReader` ve `FileWriter` sınıflarının temel kullanımı.
- **Örnek**: Bir metin dosyasına yazma ve okuma.
- **Uygulama** *(kod örneği)*:
  ```java
  // Yazma
  try (FileWriter writer = new FileWriter("metin.txt")) {
      writer.write("Merhaba, Java dosya işlemleri!");
  } catch (IOException e) {
      e.printStackTrace();
  }

  // Okuma
  try (FileReader reader = new FileReader("metin.txt")) {
      int karakter;
      while ((karakter = reader.read()) != -1) {
          System.out.print((char) karakter);
      }
  } catch (IOException e) {
      e.printStackTrace();
  }
  ```
- **Değerlendirme**: `FileReader`/`FileWriter`’ın performans sorunları (her karakter için sistem çağrısı) ve tamponlama ihtiyacı.

## 3. Tamponlu Okuma/Yazma: `BufferedReader` ve `BufferedWriter`
- **Kavram**: Tamponlama (buffering) ile performans artışı; `BufferedReader` ve `BufferedWriter` sınıflarının `FileReader`/`FileWriter` ile birlikte kullanımı.
- **Örnek**: Büyük bir metin dosyasını satır satır okuma ve yazma.
- **Uygulama** *(kod örneği)*:
  ```java
  // Yazma
  try (BufferedWriter writer = new BufferedWriter(new FileWriter("buyuk_metin.txt"))) {
      writer.write("Birinci satır");
      writer.newLine();
      writer.write("İkinci satır");
  } catch (IOException e) {
      e.printStackTrace();
  }

  // Okuma
  try (BufferedReader reader = new BufferedReader(new FileReader("buyuk_metin.txt"))) {
      String satir;
      while ((satir = reader.readLine()) != null) {
          System.out.println(satir);
      }
  } catch (IOException e) {
      e.printStackTrace();
  }
  ```
- **Değerlendirme**: Tamponlu sınıfların performans avantajı ve `readLine()` gibi kolaylık sağlayan metotlar.

## 4. Modern Dosya İşlemleri: NIO.2 (`Path`, `Files`, `BufferedReader/BufferedWriter`)
- **Kavram**: Java NIO.2 (New I/O 2) paketi (`java.nio.file`) ile daha esnek, hatalara dayanıklı ve özellikli dosya işlemleri; `Path` ve `Files` sınıfları.
- **Örnek**: `Files.readString()` ve `Files.writeString()` ile basit metin işlemleri; `Files.list()` ile dizin içeriğini listeleme.
- **Uygulama** *(kod örneği)*:
  ```java
  Path dosyaYolu = Paths.get("nio_ornek.txt");

  // Yazma
  try {
      Files.writeString(dosyaYolu, "NIO.2 ile yazma işlemi", StandardOpenOption.CREATE);
  } catch (IOException e) {
      e.printStackTrace();
  }

  // Okuma
  try {
      String icerik = Files.readString(dosyaYolu);
      System.out.println("Dosya içeriği: " + icerik);
  } catch (IOException e) {
      e.printStackTrace();
  }

  // Dizin listeleme
  try (Stream<Path> stream = Files.list(Paths.get("."))) {
      stream.forEach(System.out::println);
  } catch (IOException e) {
      e.printStackTrace();
  }
  ```
- **Değerlendirme**: NIO.2’nin geleneksel IO’ya göre avantajları (daha kısa kod, hata yönetimi, sembolik link desteği) ve `try-with-resources` kullanımı.

## 5. Nesne Kalıcılığı: Serileştirme (Serialization)
- **Kavram**: Java nesnelerini byte akışına dönüştürme (serileştirme) ve byte akışından nesne oluşturma (deserileştirme); `Serializable` arayüzü, `ObjectOutputStream`/`ObjectInputStream`.
- **Örnek**: Bir `Person` sınıfını serileştirme ve dosyaya yazma; ardından dosyadan okuyup nesneye dönüştürme.
- **Uygulama** *(kod örneği)*:
  ```java
  // Person sınıfı
  class Person implements Serializable {
      private static final long serialVersionUID = 1L;
      String ad;
      int yas;

      Person(String ad, int yas) {
          this.ad = ad;
          this.yas = yas;
      }
  }

  // Serileştirme
  try (ObjectOutputStream oos = new ObjectOutputStream(new FileOutputStream("kisi.dat"))) {
      Person kisi = new Person("Ali", 30);
      oos.writeObject(kisi);
      System.out.println("Nesne dosyaya yazıldı.");
  } catch (IOException e) {
      e.printStackTrace();
  }

  // Deserileştirme
  try (ObjectInputStream ois = new ObjectInputStream(new FileInputStream("kisi.dat"))) {
      Person okunanKisi = (Person) ois.readObject();
      System.out.println("Ad: " + okunanKisi.ad + ", Yaş: " + okunanKisi.yas);
  } catch (IOException | ClassNotFoundException e) {
      e.printStackTrace();
  }
  ```
- **Değerlendirme**: Serileştirmenin güvenlik riskleri (geçersiz veri), `serialVersionUID` önemi ve geçici (`transient`) alanlar; alternatif olarak JSON/XML kullanımı.
