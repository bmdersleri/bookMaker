# Java Programının Temel Yapısı, Değişkenler ve Veri Tipleri

## 1. Java Programının Temel Bileşenleri
### 1.1 Class Yapısı ve Tanımı
- Class bildirimi sözdizimi
- Class adlandırma kuralları (PascalCase)
- Class scope'u ve süslü parantezler
- ```[KOD]``` public class Ornek { ... }

### 1.2 main Metodu ve Programın Başlangıç Noktası
- main metodunun imzası (public static void main(String[] args))
- JVM'in main metodunu çağırma mekanizması
- args parametresinin anlamı
- ```[KOD]``` public static void main(String[] args) { ... }

### 1.3 Kod Blokları ve Kapsam (Scope)
- Blok tanımı ve süslü parantez kullanımı
- İç içe blok yapıları
- Blokların hiyerarşisi ve erişim kuralları
- ```[KOD]``` { int x = 5; { int y = 10; } }

## 2. Yorum Satırları ve Kod Dokümantasyonu
### 2.1 Tek Satırlık Yorumlar
- // kullanımı ve özellikleri
- Debug amaçlı kullanım
- ```[KOD]``` // Bu bir tek satırlık yorumdur

### 2.2 Çok Satırlı Yorumlar
- /* ... */ sözdizimi
- Kod içinde geçici devre dışı bırakma
- ```[KOD]``` /* Çok satırlı yorum başlangıcı ... sonu */

### 2.3 Javadoc Yorumları
- /** ... */ yapısı
- @param, @return, @author gibi etiketler
- ```[KOD]``` /** * @param sayi * @return kare değeri */

## 3. Çıktı Komutları ve Konsol İşlemleri
### 3.1 System.out.println() Metodu
- Çıktı verme ve satır atlama
- String birleştirme operatörü (+)
- ```[KOD]``` System.out.println("Merhaba Dünya");

### 3.2 System.out.print() Metodu
- println() ve print() arasındaki fark
- Yan yana çıktı verme
- ```[KOD]``` System.out.print("Java "); System.out.print("Programlama");

### 3.3 Escape Karakterleri ve Formatlı Çıktı
- \n, \t, \\" gibi özel karakterler
- System.out.printf() kullanımı
- ```[KOD]``` System.out.println("Satır1\nSatır2\tSekmeli");

## 4. Değişkenler ve Veri Tipleri
### 4.1 Değişken Tanımlama ve Atama
- Değişken bildirimi sözdizimi (tip ad = değer;)
- Değişken adlandırma kuralları (camelCase)
- ```[KOD]``` int yas = 25; String isim = "Ali";

### 4.2 İlkel (Primitive) Veri Tipleri
- Tam sayı tipleri: byte, short, int, long
- Ondalıklı tipler: float, double
- Karakter tipi: char
- Mantıksal tip: boolean
- ```[KOD]``` byte b = 127; double pi = 3.14; boolean aktif = true;

### 4.3 Referans Veri Tipleri
- String sınıfı ve özellikleri
- Dizi (Array) tanımlama
- ```[KOD]``` String metin = "Java"; int[] sayilar = {1, 2, 3};

## 5. Uygulama ve Değerlendirme
### 5.1 Örnek Program Geliştirme
- Tüm kavramları içeren bütünleşik örnek
- Adım adım kod yazma süreci
- ```[KOD]``` public class IlkProgram { public static void main(String[] args) { // Değişken tanımlamaları int ogrenciSayisi = 150; String okulAdi = "Teknik Lise"; // Çıktı işlemleri System.out.println("Okul: " + okulAdi); System.out.println("Öğrenci Sayısı: " + ogrenciSayisi); /* Hesaplama yapılıyor */ double ortalama = 85.5; System.out.printf("Not ortalaması: %.1f%n", ortalama); } }

### 5.2 Sık Yapılan Hatalar ve Çözümleri
- Class ve main metodu yazım hataları
- Veri tipi uyuşmazlıkları
- Değişken isimlendirme hataları
- Noktalı virgül unutma
- ```[KOD]``` // Hatalı kullanım örnekleri ve düzeltmeleri

- Farklı veri tipleriyle değişken tanımlama
- Çıktı formatlama alıştırmaları
