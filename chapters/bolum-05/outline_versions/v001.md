# Karar Yapıları ve Koşullu Programlama

## 1. Koşullu Programlamaya Giriş
### 1.1. Koşullu Programlama Nedir?
- Program akışının koşullara göre yönlendirilmesi
- Gerçek hayattan karar verme örnekleri (örneğin: yağmur yağıyorsa şemsiye al)
- Kod bloğu seçiminin önemi

### 1.2. Algoritmik Düşüncede Karar Yapılarının Yeri
- Akış diyagramları ile karar noktalarının gösterimi
- Koşul ifadelerinin temel mantığı
- Boolean (doğru/yanlış) değer kavramı

### 1.3. Kod Örneği: Basit Koşul Akışı
```python
# Kod örneği: Koşulsuz akış vs koşullu akış karşılaştırması
print("Her zaman çalışır")
# if koşulu: sadece belirli durumda çalışır
```

## 2. Karşılaştırma Operatörleri
### 2.1. Temel Karşılaştırma Operatörleri
- Eşitlik (`==`) ve eşitsizlik (`!=`) operatörleri
- Büyüktür (`>`) ve küçüktür (`<`) operatörleri
- Büyük eşit (`>=`) ve küçük eşit (`<=`) operatörleri

### 2.2. Karşılaştırma Operatörlerinin Kullanımı
- Sayısal değerlerle karşılaştırma
- String (metin) karşılaştırmaları
- Farklı veri tiplerinde karşılaştırma davranışları

### 2.3. Kod Örneği: Karşılaştırma Operatörleri
```python
# Kod örneği: Tüm karşılaştırma operatörlerinin kullanımı
yas = 18
print(yas == 18)  # True
print(yas != 20)  # True
print(yas > 15)   # True
print(yas < 20)   # True
print(yas >= 18)  # True
print(yas <= 17)  # False
```

### 2.4. Uygulama: Kullanıcı Girişi ile Karşılaştırma
```python
# Kod örneği: Kullanıcıdan alınan değerleri karşılaştırma
kullanici_yas = int(input("Yaşınızı girin: "))
yetiskin_siniri = 18
print("Yetişkin misiniz?", kullanici_yas >= yetiskin_siniri)
```

## 3. if ve else Yapıları
### 3.1. if Yapısının Temel Kullanımı
- `if` anahtar kelimesi ve sözdizimi
- Girintileme (indentation) kuralları
- Tek koşullu karar yapısı

### 3.2. if-else Yapısı
- `else` anahtar kelimesi ve alternatif akış
- İki yönlü karar mekanizması
- Koşul sağlandığında ve sağlanmadığında çalışan kod blokları

### 3.3. Kod Örneği: if-else Kullanımı
```python
# Kod örneği: if-else yapısı ile sınav notu değerlendirme
not = 75
if not >= 60:
    print("Geçtiniz!")
else:
    print("Kaldınız!")
```

### 3.4. İç İçe if Yapıları (Nested if)
- Bir if bloğu içinde başka bir if kullanımı
- Dikkat edilmesi gereken girintileme kuralları
- Karmaşık karar ağaçları oluşturma

### 3.5. Uygulama: Hesap Makinesi Doğrulama
```python
# Kod örneği: Kullanıcı giriş doğrulama sistemi
kullanici_adi = input("Kullanıcı adı: ")
sifre = input("Şifre: ")

if kullanici_adi == "admin":
    if sifre == "1234":
        print("Giriş başarılı!")
    else:
        print("Hatalı şifre!")
else:
    print("Kullanıcı bulunamadı!")
```

## 4. else-if (elif) Yapısı
### 4.1. elif Anahtar Kelimesi
- Çoklu koşul kontrolü ihtiyacı
- `elif` sözdizimi ve kullanım amacı
- if-elif-else zincirinin mantığı

### 4.2. elif ile Çoklu Koşul Yönetimi
- Koşulların sıralı kontrolü
- İlk sağlanan koşulun çalıştırılması
- else bloğunun opsiyonel olması

### 4.3. Kod Örneği: elif ile Not Sistemi
```python
# Kod örneği: elif ile harf notu hesaplama
puan = 85

if puan >= 90:
    harf = "AA"
elif puan >= 80:
    harf = "BA"
elif puan >= 70:
    harf = "BB"
elif puan >= 60:
    harf = "CB"
elif puan >= 50:
    harf = "CC"
else:
    harf = "FF"

print(f"Notunuz: {harf}")
```

### 4.4. Uygulama: Günlük Aktivite Planlayıcı
```python
# Kod örneği: Hava durumuna göre aktivite önerme
hava_durumu = input("Hava nasıl? (güneşli/yağmurlu/karli/bulutlu): ")

if hava_durumu == "güneşli":
    print("Plaja gidebilirsiniz!")
elif hava_durumu == "yağmurlu":
    print("Sinemaya gidebilirsiniz!")
elif hava_durumu == "karli":
    print("Kayak yapmaya gidebilirsiniz!")
elif hava_durumu == "bulutlu":
    print("Parkta yürüyüş yapabilirsiniz!")
else:
    print("Geçersiz hava durumu girdiniz!")
```

## 5. Mantıksal Koşullar (AND, OR, NOT)
### 5.1. AND Operatörü
- Tüm koşulların doğru olması gerektiği durumlar
- `and` anahtar kelimesi kullanımı
- Kısa devre (short-circuit) değerlendirmesi

### 5.2. OR Operatörü
- En az bir koşulun doğru olması yeterli
- `or` anahtar kelimesi kullanımı
- Kısa devre değerlendirmesi

### 5.3. NOT Operatörü
- Koşulun tersini alma
- `not` anahtar kelimesi kullanımı
- Boolean değerleri ters çevirme

### 5.4. Kod Örneği: Mantıksal Operatörler
```python
# Kod örneği: Mantıksal operatörlerin kullanımı
yas = 25
ehliyet_var = True

# AND ile kontrol
if yas >= 18 and ehliyet_var:
    print("Araba kullanabilirsiniz!")

# OR ile kontrol
if yas < 18 or not ehliyet_var:
    print("Araba kullanamazsınız!")

# NOT ile kontrol
if not ehliyet_var:
    print("Ehliyet almanız gerekiyor!")
```

### 5.5. Karmaşık Mantıksal İfadeler
- Birden fazla mantıksal operatörün birleştirilmesi
- Parantez kullanımı ve öncelik sırası
- Okunabilirlik için düzenleme

### 5.6. Uygulama: Kapsamlı Kullanıcı Doğrulama
```python
# Kod örneği: Karmaşık mantıksal koşullar ile kullanıcı doğrulama
kullanici_tipi = input("Kullanıcı tipi (admin/kullanici): ")
sifre = input("Şifre: ")
otp_kodu = input("OTP Kodu: ")

if (kullanici_tipi == "admin" and sifre == "admin123") or \
   (kullanici_tipi == "kullanici" and sifre == "user123" and otp_kodu == "123456"):
    print("Giriş başarılı!")
else:
    print("Giriş başarısız!")
```

## 6. switch-case Yapısı
### 6.1. switch-case Mantığına Giriş
- Çoklu durum kontrolü için alternatif yapı
- if-elif zincirine göre avantajları
- Python'da switch-case olmaması ve çözüm yolları

### 6.2. Python'da match-case Yapısı (Python 3.10+)
- `match` ve `case` anahtar kelimeleri
- Sözdizimi ve kullanım kuralları
- Varsayılan durum (`case _`)

### 6.3. Kod Örneği: match-case Kullanımı
```python
# Kod örneği: match-case ile gün adı bulma
gun_numarasi = int(input("Gün numarasını girin (1-7): "))

match gun_numarasi:
    case 1:
        print("Pazartesi")
    case 2:
        print("Salı")
    case 3:
        print("Çarşamba")
    case 4:
        print("Perşembe")
    case 5:
        print("Cuma")
    case 6:
        print("Cumartesi")
    case 7:
        print("Pazar")
    case _:
        print("Geçersiz gün numarası!")
```

### 6.4. match-case ile Desen Eşleme
- Birden fazla değer eşleme (`case 1 | 2 | 3`)
- Koşullu desen eşleme (`case x if x > 5`)
- Sınıf ve veri yapıları ile desen eşleme

### 6.5. Uygulama: Hesap Makinesi
```python
# Kod örneği: match-case ile basit hesap makinesi
sayi1 = float(input("Birinci sayı: "))
islem = input("İşlem (+, -, *, /): ")
sayi2 = float(input("İkinci sayı: "))

match islem:
    case "+":
        print(f"Sonuç: {sayi1 + sayi2}")
    case "-":
        print(f"Sonuç: {sayi1 - sayi2}")
    case "*":
        print(f"Sonuç: {sayi1 * sayi2}")
    case "/":
        if sayi2 != 0:
            print(f"Sonuç: {sayi1 / sayi2}")
        else:
            print("Sıfıra bölme hatası!")
    case _:
        print("Geçersiz işlem!")
```

### 7.3. Uygulama Projesi: ATM Simülasyonu
```python
# Kod örneği: Tüm konuları kapsayan ATM simülasyonu
bakiye = 1000
deneme_sayisi = 3

print("ATM'ye hoş geldiniz!")

while deneme_sayisi > 0:
    sifre = input("Şifrenizi girin: ")
    
    if sifre == "1234":
        print("Giriş başarılı!")
        
        while True:
            print("\n1. Bakiye Sorgula")
            print("2. Para Çek")
            print("3. Para Yatır")
            print("4. Çıkış")
            
            secim = input("Seçiminiz: ")
            
            match secim:
                case "1":
                    print(f"Bakiyeniz: {bakiye} TL")
                case "2":
                    miktar = float(input("Çekilecek miktar: "))
                    if miktar > 0 and miktar <= bakiye:
                        bakiye -= miktar
                        print(f"{miktar} TL çekildi. Kalan bakiye: {bakiye}")
                    elif miktar > bakiye:
                        print("Yetersiz bakiye!")
                    else:
                        print("Geçersiz miktar!")
                case "3":
                    miktar = float(input("Yatırılacak miktar: "))
                    if miktar > 0:
                        bakiye += miktar
                        print(f"{miktar} TL yatırıldı. Güncel bakiye: {bakiye}")
                    else:
                        print("Geçersiz miktar!")
                case "4":
                    print("Çıkış yapılıyor...")
                    break
                case _:
                    print("Geçersiz seçim!")
        break
    else:
        deneme_sayisi -= 1
        if deneme_sayisi > 0:
            print(f"Hatalı şifre! Kalan deneme: {deneme_sayisi}")
        else:
            print("Kartınız bloke oldu!")
```
