## Math Sınıfının Temel Metotları ve Kullanımı
- `Math` sınıfının statik yapısı ve import gereksinimi
- Temel aritmetik metotlar:
  - `abs()`: Mutlak değer işlemleri
  - `max()` ve `min()`: Karşılaştırma işlemleri
  - `pow()`: Üs alma işlemleri
  - `sqrt()`: Karekök hesaplama
- Yuvarlama metotları:
  - `ceil()`: Yukarı yuvarlama
  - `floor()`: Aşağı yuvarlama
  - `round()`: En yakın tamsayıya yuvarlama
- Trigonometrik ve logaritmik metotlar:
  - `sin()`, `cos()`, `tan()`: Trigonometrik hesaplamalar
  - `log()`, `log10()`: Logaritmik hesaplamalar

**Kod örneği:** [Math sınıfı temel metotları kullanımı]
```java
// Örnek kod: Math sınıfı metotları
double sayi = -5.7;
System.out.println("Mutlak değer: " + Math.abs(sayi));
System.out.println("Yukarı yuvarla: " + Math.ceil(sayi));
System.out.println("Aşağı yuvarla: " + Math.floor(sayi));
System.out.println("En yakın: " + Math.round(sayi));
System.out.println("2^3 = " + Math.pow(2, 3));
System.out.println("25'in karekökü = " + Math.sqrt(25));
```

**Kod örneği:** [Matematiksel sabitler ve trigonometri]
```java
// Örnek kod: Pi ve trigonometrik işlemler
double aci = 45;
double radyan = Math.toRadians(aci);
System.out.println("Pi sayısı: " + Math.PI);
System.out.println("Sin 45°: " + Math.sin(radyan));
System.out.println("Cos 45°: " + Math.cos(radyan));
```

## Rastgele Sayı Üretimi Temelleri
- `Math.random()` metodu ve çalışma prensibi
- 0.0 ile 1.0 arasında ondalıklı sayı üretimi
- Belirli aralıkta rastgele sayı üretme formülü: `(int)(Math.random() * (max - min + 1)) + min`
- `Random` sınıfına giriş:
  - `nextInt()`, `nextDouble()`, `nextBoolean()` metotları
  - Seed kavramı ve deterministik rastgelelik
- `ThreadLocalRandom` sınıfı (çoklu iş parçacığı ortamları için)

**Kod örneği:** [Math.random() ile rastgele sayı üretimi]
```java
// Örnek kod: Math.random() kullanımı
System.out.println("0-1 arası rastgele: " + Math.random());
System.out.println("1-10 arası rastgele tamsayı: " + 
    (int)(Math.random() * 10 + 1));
System.out.println("50-100 arası rastgele tamsayı: " + 
    (int)(Math.random() * 51 + 50));
```

**Kod örneği:** [Random sınıfı kullanımı]
```java
// Örnek kod: Random sınıfı ile rastgele üretim
import java.util.Random;
Random rastgele = new Random();
System.out.println("Rastgele tamsayı: " + rastgele.nextInt());
System.out.println("0-100 arası: " + rastgele.nextInt(101));
System.out.println("Rastgele ondalık: " + rastgele.nextDouble());
System.out.println("Rastgele boolean: " + rastgele.nextBoolean());
```

## Rastgele Sayı Üretiminde İleri Teknikler
- Belirli dağılımlara uygun rastgele sayı üretimi
- Gaussian (normal) dağılım: `nextGaussian()` metodu
- Rastgele karakter ve dizi üretimi
- Şifreleme amaçlı güvenli rastgele sayı üretimi: `SecureRandom` sınıfı
- Rastgele sayı üretiminde performans optimizasyonu
- Seed yönetimi ve tekrarlanabilir rastgelelik

**Kod örneği:** [Normal dağılım ve SecureRandom kullanımı]
```java
// Örnek kod: İleri rastgele teknikler
import java.util.Random;
import java.security.SecureRandom;

Random rastgele = new Random();
SecureRandom guvenli = new SecureRandom();

System.out.println("Normal dağılım: " + rastgele.nextGaussian());

// Güvenli rastgele sayı
System.out.println("Güvenli rastgele: " + guvenli.nextInt(100));

// Rastgele karakter üretimi
char rastgeleHarf = (char)('A' + rastgele.nextInt(26));
System.out.println("Rastgele harf: " + rastgeleHarf);
```

**Kod örneği:** [Seed ile tekrarlanabilir rastgelelik]
```java
// Örnek kod: Seed kullanımı
Random r1 = new Random(42);
Random r2 = new Random(42);

System.out.println("r1: " + r1.nextInt(100));
System.out.println("r2: " + r2.nextInt(100));
System.out.println("r1: " + r1.nextInt(100));
System.out.println("r2: " + r2.nextInt(100));
// Aynı seed ile aynı sayılar üretilir
```

## Matematiksel Problem Çözme Yaklaşımları
- Problem analizi ve çözüm stratejisi geliştirme
- Matematiksel modelleme teknikleri
- Sayısal yöntemler:
  - Yaklaşık hesaplamalar ve hata toleransı
  - Newton-Raphson metodu ile kök bulma
  - Monte Carlo simülasyonu
- Optimizasyon problemleri:
  - En küçük/en büyük değer bulma
  - Brute-force yaklaşımı
  - Dinamik programlama temelleri
- Gerçek hayat problemlerine uygulama:
  - İstatistiksel hesaplamalar (ortalama, varyans, standart sapma)
  - Geometrik hesaplamalar (alan, hacim)
  - Finansal hesaplamalar (faiz, amortisman)

**Kod örneği:** [Monte Carlo ile Pi hesaplama]
```java
// Örnek kod: Monte Carlo Pi hesaplama
public class MonteCarloPi {
    public static void main(String[] args) {
        int toplamNokta = 1000000;
        int icDaire = 0;
        
        for (int i = 0; i < toplamNokta; i++) {
            double x = Math.random() * 2 - 1;
            double y = Math.random() * 2 - 1;
            if (x * x + y * y <= 1) {
                icDaire++;
            }
        }
        
        double pi = 4.0 * icDaire / toplamNokta;
        System.out.println("Hesaplanan Pi: " + pi);
        System.out.println("Gerçek Pi: " + Math.PI);
        System.out.println("Hata: " + Math.abs(pi - Math.PI));
    }
}
```

**Kod örneği:** [İstatistiksel hesaplamalar]
```java
// Örnek kod: Dizi üzerinde istatistiksel işlemler
import java.util.Random;

public class IstatistikHesaplamalar {
    public static void main(String[] args) {
        int[] sayilar = {12, 45, 67, 23, 89, 34, 56};
        
        // Ortalama
        double toplam = 0;
        for (int sayi : sayilar) {
            toplam += sayi;
        }
        double ortalama = toplam / sayilar.length;
        System.out.println("Ortalama: " + ortalama);
        
        // Standart sapma
        double karelerToplami = 0;
        for (int sayi : sayilar) {
            karelerToplami += Math.pow(sayi - ortalama, 2);
        }
        double standartSapma = Math.sqrt(karelerToplami / sayilar.length);
        System.out.println("Standart Sapma: " + standartSapma);
        
        // En büyük ve en küçük değer
        int enBuyuk = sayilar[0];
        int enKucuk = sayilar[0];
        for (int sayi : sayilar) {
            enBuyuk = Math.max(enBuyuk, sayi);
            enKucuk = Math.min(enKucuk, sayi);
        }
        System.out.println("En büyük: " + enBuyuk + ", En küçük: " + enKucuk);
    }
}
```

**Kod örneği:** [Zar atma simülasyonu]
```java
// Örnek kod: Zar atma simülasyonu
import java.util.Random;

public class ZarSimulasyonu {
    public static void main(String[] args) {
        Random rastgele = new Random();
        int[] zarSonuclari = new int[6];
        
        // 1000 kez zar at
        for (int i = 0; i < 1000; i++) {
            int zar = rastgele.nextInt(6) + 1;
            zarSonuclari[zar - 1]++;
        }
        
        // Sonuçları göster
        for (int i = 0; i < zarSonuclari.length; i++) {
            double yuzde = (double)zarSonuclari[i] / 1000 * 100;
            System.out.printf("%d: %d kez (%.1f%%)\n", 
                i+1, zarSonuclari[i], yuzde);
        }
    }
}
```

**Kod örneği:** [Rastgele şifre oluşturucu]
```java
// Örnek kod: Rastgele şifre oluşturucu
import java.util.Random;
import java.security.SecureRandom;

public class SifreOlusturucu {
    public static String sifreUret(int uzunluk) {
        SecureRandom rastgele = new SecureRandom();
        String karakterler = "ABCDEFGHIJKLMNOPQRSTUVWXYZ" +
                           "abcdefghijklmnopqrstuvwxyz" +
                           "0123456789!@#$%^&*";
        StringBuilder sifre = new StringBuilder();
        
        for (int i = 0; i < uzunluk; i++) {
            int index = rastgele.nextInt(karakterler.length());
            sifre.append(karakterler.charAt(index));
        }
        
        return sifre.toString();
    }
    
    public static void main(String[] args) {
        System.out.println("8 karakterli şifre: " + sifreUret(8));
        System.out.println("12 karakterli şifre: " + sifreUret(12));
        System.out.println("16 karakterli şifre: " + sifreUret(16));
    }
}
```
